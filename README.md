# Aerospace-Cost-Management-tool
Project Description: Aerospace Cost Management Tool
Summary: Developed a financial tool for an aerospace project, focused on cost control, analysis, and forecasting.

Key Features:

Data Quality & Enrichment: Cleaned raw cost data (handling duplicates/nulls) and calculated performance metrics like Cost Variance and Cost Efficiency.

Reporting & Analysis: Provided spending visibility by Department, Project Phase, Vendor, and over Time (Monthly Trends).

Statistical Modeling: Applied Regression for budget-actual comparison and Hypothesis Tests (T-Test, Z-Test, ANOVA) to validate cost differences and perform Cost Forecasting.

Skills: Data cleaning, financial modeling, statistical analysis, and business intelligence (BI) reporting.
